# CodeShade I.D.E

CodeShade I.D.E is a lightweight desktop code editor built using Python and Tkinter.  
It is designed for focused, distraction-free programming with basic IDE features for Python and C++.

This project was built incrementally while learning desktop application architecture, file handling, and process execution in Python.
